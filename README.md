# vite-template-sky

An opinionated vite starter template for vue or react with a cli to generate template project.

## Usage

Using npm

```sh
npx @sky-fly/vite-template
```

Using yarn

```sh
yarn dlx @sky-fly/vite-template
```

Using pnpm

```sh
pnpm dlx @sky-fly/vite-template
```
